# F1 Data Warehouse

This project builds a local data warehouse using Formula 1 historical data.

## Goal
- Ingest public F1 data
- Store raw data in PostgreSQL
- Transform it into analytics-ready tables

## Tech Stack
- Python
- PostgreSQL
- Docker
- dbt (planned)

## Data Source
- Ergast API (public Formula 1 data)
